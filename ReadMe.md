The key to control game:

 ----------------------------------------------------
|           Key          |         Be used to        |
 ----------------------------------------------------
| Up arrow               | Jump						 |
| Down arrow             | Stoop					 |
| P                      | Pause					 |
| R                      | Restart when game over	 |
| E                      | Exit game when game over	 |
